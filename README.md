-Overview

This project focuses on predicting stock price trends of PT ASRI using the Linear Regression algorithm based on historical stock data.

-Key Highlights
1. Stock trend analysis (2020–2023)
2. Machine Learning model (Linear Regression)
3. Actual vs Predicted visualization
4. Data-driven insights

-Tools & Technologies
Python, Pandas, NumPy, Matplotlib, Scikit-learn
